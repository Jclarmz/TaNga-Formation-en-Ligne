Résumé du cours

Windows est un système d'exploitation de Microsoft conçu pour fonctionner sur les ordinateurs de bureau, les ordinateurs portables, les tablettes et les téléphones intelligents. Il offre une variété de fonctionnalités et d'outils pour les utilisateurs.
Ce système permet de se familiariser avec son ordinateur.
